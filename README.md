# 🚀 Featured Project

## 📱 Mini Social Media Platform

A mini social media application built using Node.js, Express.js, MongoDB, JWT Authentication, and EJS.

### ✨ Features
- User Authentication
- Create & Like Posts
- JWT Authentication
- Cookies Handling
- MongoDB Database Integration

---

## 🛠️ Run Locally

```bash
git clone https://github.com/Parishmitabanik/MiniSocialMediaPlatform.git
cd MiniSocialMediaPlatform
npm install
nodemon app.js
```

Then open:

```text
http://localhost:3000
```
